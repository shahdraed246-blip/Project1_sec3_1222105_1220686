# Vehicle Delivery Route Optimizer

Assigns packages to vehicles and finds the shortest delivery routes using **Simulated Annealing** and **Genetic Algorithm**.

## How to Run
1. Install dependencies:
   pip install matplotlib numpy
2. Run:
   1222105_1220686.py
3. Enter vehicle capacities, package details, and choose an algorithm.
